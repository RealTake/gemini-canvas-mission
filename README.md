# gemini-canvas-mission

Gemini Canvas 미션 결과물을 관리하는 저장소입니다.

## 목적

- Gemini Canvas로 제작한 웹앱 결과물을 기록하고 공유합니다.
- 앱별 요구사항, 배포 링크, 회고를 일관된 형식으로 남깁니다.

## 카테고리: 유틸리티 앱

[https://gemini.google.com/share/6617bf518393]

- 점심 메뉴 랜덤 추천

## 앱 이름: 우테코 회의 메이트 Pro
## 카테고리: 페어 프롬프트 릴레이
## 페어: 토리
### 배포 링크
[https://gemini.google.com/share/14e4da8d29d1]

### 이 앱을 만든 이유
- 어떤 문제/불편함을 해결하려고 했나요?
  - 우테코 생활하면서 작고 크게 회의나 미팅이 많이 있는데, 해당 내용을 효과적으로 기록할 수 있는 앱이 있으면 좋겠다고 생각했습니다.
- 이 앱을 언제, 어떻게 사용할 건가요?
  - 회의나 미팅 시작 시 이 앱을 켜서 회의 내용을 자동으로 기록할 예정입니다.

### 주요 기능
- 회의 내용 녹음 기능
- 녹음된 내용을 대화 형태로 추출해주는 기능
- 녹음된 텍스트를 요약하여 제목과 내용으로 제공하는 기능
- 녹음된 내용을 TODO 리스트로 변환하는 기능

### AI 기능 (해당되는 경우)
- 어떤 AI 기능을 추가했나요?
  - Gemini의 대화형 인터렉션 기능(오디오 분석)
  - Gemini의 텍스트 생성 기능
- AI 기능이 앱에서 어떤 역할을 하나요?
  - 녹음된 음성을 텍스트로 변환
  - 변환된 텍스트를 요약하여 제목, 내용, TODO 목록으로 적절히 변환하는 역할을 합니다.
